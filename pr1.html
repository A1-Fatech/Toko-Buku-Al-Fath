Diri Saya

Perkenalkan nama saya Fatechah Seto Jati biasa di panggil Seto. Saya lahir di Magelang tanggal 10 Agustus 2006 dan saat ini saya berusia 18 tahun. Alamat rumah saya di Dusun Tosari rt14/ rw06 Rejosari Kecamatan Pakis. Saat ini saya berkuliah di STMIK BINA PATRIA Magelang semester 2. Disini saya mengambil jurusan Teknik Informatika jenjang S1. Saya berharap dapat lulus dalam waktu 4 tahun.

Dalam kehidupan sehari-hari mungkin orang-orang mengira saya orangnya tidak serius. Saya memang suka bercanda tapi ada kalanya saat saya harus serius, saya akan serius terlebih saat dalam perkuliahan. Saya selalu ingat perjuangan kedua orangtua untuk membiayai saya, mereka selalu bekerja keras jadi saya harus belajar lebih keras untuk membanggakan orangtua. Selain sebagai mahasiswa, selayaknya hidup bermasyarakat saya harus tetap meluangkan waktu untuk mengikuti kegiatan yang ada di masyarakat. Sesibuk-sibuknya kita tetap harus meluangkan waktu karena saat kita kesusahan orang pertama yang menolong adalah tetangga terdekat kita.

Memang saat ini saya masuk jurusan Teknik Informatika, namun sebelumnya saya belum pernah mempelajari materi-materi didalamnya karena sebelumnya saya sekolah SMA. Jadi saya harus berusaha supaya tidak tertinggal dari teman-teman lainnya. Saya memilih jurusan ini karena saya berharap suatu saat nanti bisa menjadi game developer. Saya ingin membuat game sendiri dan mengembangkannya agar menghibur bagi banyak orang.
